# HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў

**рЃДрЃЋрЃћрЃџрЃљ рЃЋрЃћрЃЉ-рЃАрЃљрЃўрЃбрЃў рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃњрЃљрЃю**

## рЃарЃљ рЃљрЃарЃўрЃА HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў?
рЃЌрЃБ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃА рЃерЃћрЃ«рЃћрЃЉрЃљ рЃ░рЃЦрЃЮрЃюрЃўрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃЌрЃљрЃю, рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃЊрЃљ рЃарЃљрЃўрЃЏрЃћ рЃбрЃўрЃърЃўрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА. 

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:
- рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ Word - рЃА, Google Docs - рЃА рЃЊрЃљ рЃАрЃ«рЃЋрЃћрЃЉрЃА.
- рЃфрЃ«рЃарЃўрЃџрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ Excel - рЃА, Google Sheets рЃЊрЃљ рЃљ.рЃе.

рЃќрЃБрЃАрЃбрЃљрЃЊ рЃљрЃАрЃћрЃЌрЃўрЃЋрЃћ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃюрЃљрЃўрЃарЃАрЃљрЃ«рЃћрЃЮрЃЉрЃљрЃА рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў. рЃЕрЃЋрЃћрЃю рЃЏрЃљрЃЌ рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃА рЃЋрЃћрЃФрЃљрЃ«рЃўрЃЌ. рЃЌрЃБрЃЏрЃфрЃљ рЃЏрЃљрЃЌрЃў рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљрЃљ рЃўрЃАрЃћрЃЌрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃњрЃљрЃЋрЃ«рЃАрЃюрЃљ рЃарЃљрЃф рЃЕрЃћрЃЏрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃерЃў рЃљрЃа рЃЏрЃљрЃЦрЃЋрЃА рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў. 

рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃў рЃљрЃарЃўрЃА HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ рЃЊрЃљ рЃЏрЃўрЃўрЃдрЃЮ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃерЃћрЃюрЃЌрЃЋрЃўрЃА рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃЋрЃћрЃЉ-рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃЊрЃљрЃю (URL).

рЃЕрЃЋрЃћрЃюрЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ рЃерЃћрЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃЮрЃЌ рЃарЃЮрЃњрЃЮрЃа рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃЌрЃљрЃЋрЃљрЃЊ рЃЊрЃљрЃЋрЃўрЃгрЃДрЃЮрЃЌ рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃўрЃА рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃў. 

## рЃарЃљ рЃљрЃарЃўрЃА рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃў?
рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃў рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃљ рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃДрЃЮрЃЋрЃћрЃџрЃЊрЃдрЃўрЃБрЃарЃљрЃЊ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ рЃЊрЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ. 

рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў:
- рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃЏрЃљ рЃўрЃфрЃўрЃА рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃўрЃАрЃћрЃЋрЃћ рЃарЃЮрЃњрЃЮрЃарЃф Word - рЃЏрЃљ рЃўрЃфрЃўрЃА рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ.
- рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃЏрЃљ рЃўрЃфрЃўрЃА рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃА рЃЦрЃАрЃћрЃџрЃБрЃарЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ рЃЊрЃљ рЃЏрЃўрЃгрЃЋрЃЊрЃћрЃА рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.
- рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃЏрЃљ рЃўрЃфрЃўрЃА рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃљрЃўрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃА, рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃЮрЃА рЃЏрЃљрЃЌрЃњрЃљрЃю HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў, рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃЮрЃА рЃўрЃАрЃўрЃюрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃерЃў рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЕрЃЋрЃћрЃюрЃЮрЃЌ рЃўрЃАрЃюрЃў рЃћрЃЎрЃарЃљрЃюрЃќрЃћ. рЃљрЃЏ рЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃерЃћрЃАрЃЋрЃџрЃљрЃА рЃЋрЃћрЃФрЃљрЃ«рЃўрЃЌ.
- рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃерЃћрЃАрЃЋрЃџрЃљ рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃА рЃЋрЃћрЃџрЃерЃў рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљрЃА рЃЊрЃљ Enter рЃЎрЃџрЃљрЃЋрЃерЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃљрЃА. рЃљрЃЏ рЃЊрЃарЃЮрЃА рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃў рЃърЃЮрЃБрЃџрЃЮрЃЉрЃА рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА **рЃАрЃћрЃарЃЋрЃћрЃарЃА**, рЃБрЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃЏрЃљрЃА **рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљрЃА** рЃЊрЃљ рЃърЃљрЃАрЃБрЃ«рЃљрЃЊ рЃўрЃдрЃћрЃЉрЃА HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃА. рЃћрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃЏрЃфрЃўрЃарЃћрЃЊрЃў рЃЊрЃљрЃДрЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЕрЃљрЃюрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ.


## рЃЋрЃћрЃЉ-рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў
HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃЊрЃљ рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃў рЃўрЃЏ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃЮрЃЉрЃљрЃљ рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃћрЃю рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃў рЃўрЃЏ рЃцрЃЮрЃарЃЏрЃўрЃЌ рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃДрЃЮрЃЋрЃћрЃџрЃЊрЃдрЃўрЃБрЃарЃљрЃЊ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ. 

рЃўрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃарЃљрЃф рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃерЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃЋрЃћрЃЉ-рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў рЃ░рЃЦрЃЋрЃўрЃљ. рЃБрЃцрЃарЃЮ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃљрЃЊ рЃЎрЃў **Front-End** рЃЋрЃћрЃЉ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў. 

HTML - рЃўрЃА рЃњрЃљрЃарЃЊрЃљ рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃерЃў рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃўрЃАрЃћрЃЌ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ **CSS** (рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃљрЃЊ рЃњрЃљрЃАрЃљрЃцрЃЮрЃарЃЏрЃћрЃЉрЃџрЃљрЃЊ) рЃЊрЃљ **JavaScript** (рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџ рЃЋрЃћрЃЉ-рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃљрЃЊ рЃњрЃљрЃЊрЃљрЃАрЃљрЃЦрЃфрЃћрЃЋрЃљрЃЊ).

рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ HTML, CSS рЃЊрЃљ JavaSript рЃљрЃарЃўрЃА рЃўрЃА рЃўрЃљрЃарЃљрЃдрЃћрЃЉрЃў рЃарЃљрЃќрЃћрЃф рЃЊрЃњрЃљрЃА рЃЊрЃљ рЃерЃћрЃюрЃЊрЃћрЃЉрЃљ Front-End рЃЋрЃћрЃЉ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў. рЃЏрЃљрЃЌрЃќрЃћ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃЌрЃљрЃЋрЃћрЃЉрЃерЃў рЃЋрЃўрЃАрЃљрЃБрЃЉрЃарЃћрЃЉрЃЌ рЃЏрЃљрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃарЃљрЃф HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃА рЃЊрЃљрЃЋрЃцрЃљрЃарЃљрЃЋрЃЌ.

## HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
import Link from 'next/link'

рЃарЃЮрЃњрЃЮрЃарЃф рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў, HTML - рЃўрЃф рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ.

рЃАрЃўрЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃерЃў, рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃљрЃЊрЃњрЃўрЃџрЃљрЃА рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃцрЃљрЃўрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ hello.html рЃЊрЃљ рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ [рЃЎрЃЮрЃЊрЃўрЃА рЃћрЃЊрЃўрЃбрЃЮрЃарЃўрЃА]("/tools/code-editor") рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ рЃљрЃАрЃћрЃЌрЃў рЃбрЃћрЃЦрЃАрЃбрЃў "Hello, World!".

рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў рЃљрЃАрЃћ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃЮрЃЊрЃћрЃА:
```html filename="hello.html"
Hello, World!
```

## HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃерЃў
рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃў рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃерЃўрЃљ рЃЕрЃљрЃгрЃћрЃарЃўрЃџрЃў. рЃАрЃљрЃЦрЃарЃЌрЃЋрЃћрЃџрЃЮрЃерЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃ«рЃерЃўрЃарЃљрЃЊ Chrome рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃА рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃЊрЃљ Front-End - рЃўрЃА рЃАрЃгрЃљрЃЋрЃџрЃўрЃАрЃљрЃА  [Chrome](https://www.google.com/chrome/) - рЃўрЃА рЃљрЃю [Brave](https://brave.com) - рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃА рЃњрЃўрЃарЃЕрЃћрЃЋрЃЌ.

`hello.html` рЃцрЃљрЃўрЃџрЃќрЃћ рЃЮрЃарЃ»рЃћрЃа рЃЊрЃљрЃЎрЃџрЃўрЃЎрЃћрЃЉрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЏрЃўрЃћрЃа рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃњрЃљрЃўрЃ«рЃАрЃюрЃћрЃЉрЃљ Browser - рЃўрЃА рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃерЃў.

рЃерЃћрЃЊрЃћрЃњрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЊрЃљрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃўрЃЌ рЃљрЃАрЃћрЃЌрЃў:

рЃњрЃўрЃџрЃЮрЃфрЃљрЃЋ! ­ЪјЅ
рЃерЃћрЃю рЃБрЃЎрЃЋрЃћ рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃерЃћрЃюрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў. 

рЃарЃљ рЃЌрЃЦрЃЏрЃљ рЃБрЃюрЃЊрЃљ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃњрЃљрЃфрЃўрЃџрЃћрЃЉрЃўрЃЌ рЃЉрЃћрЃЋрЃарЃў рЃЊрЃћрЃбрЃљрЃџрЃўрЃАрЃљ рЃЊрЃљ рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃАрЃњрЃљрЃю рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃерЃћрЃЊрЃњрЃћрЃЉрЃЮрЃЊрЃћрЃА. рЃљрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ рЃЌрЃљрЃЋрЃћрЃЉрЃерЃў рЃњрЃљрЃЋрЃћрЃфрЃюрЃЮрЃЉрЃўрЃЌ.
