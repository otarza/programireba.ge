# HTML - ის სახელმძღვანელო

**HTML - არის ვებ-გვერდების, დოკუმენტების, აპლიკაციების და მათში მოთავსებული კონტენტის სტრუქტურის შესაქმნელი იარაღი.**

## რა არის HTML - ის სახელმძღვანელო?
ეს არის ტექსტის ფორმატით მოწოდებული HTML - ის სასწავლო გაკვეთილები. შეგიძლია გამოიყენო როგორც წიგნი რომელსაც თავიდან ბოლომდე გაყვები და დაეუფლები HTML გვერდების სტრუქტურის შექმნის ფუნდამენტურ პრინციპებს. 

ამ რესურსის გამოყენება ასევე შეგიძლია როგორც ყოველდღიური ცოდნის ბაზა. როცა რაიმეს გახსენება ან დაზუსტება გსურს მუშაობის პროცესში.
