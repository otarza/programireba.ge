# HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ
**HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.**

## DOCTYPE html
рЃўрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃарЃЮрЃЏ рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃў рЃЏрЃўрЃ«рЃЋрЃЊрЃћрЃА рЃарЃљ рЃбрЃўрЃърЃўрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃЌрЃљрЃю рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃАрЃљрЃЦрЃЏрЃћ, рЃљрЃЏрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃЏрЃљрЃА рЃарЃљрЃЏрЃћрЃюрЃљрЃўрЃарЃљрЃЊ рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃЋрЃљрЃюрЃўрЃерЃюрЃЮрЃЌ. HTML - рЃерЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃбрЃћрЃњрЃў рЃарЃЮрЃЏрЃџрЃўрЃЌрЃљрЃф рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃљрЃЋрЃдрЃгрЃћрЃарЃЮрЃЌ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃбрЃўрЃърЃў рЃЊрЃљ рЃўрЃА рЃљрЃАрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃљ `<!DOCTYPE html>`.

## html рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў
рЃЏрЃљрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃарЃљрЃф рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃбрЃўрЃърЃА рЃњрЃљрЃюрЃЋрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃЌ рЃЊрЃарЃЮрЃљ рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ html рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў. рЃћрЃА рЃўрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃДрЃЋрЃћрЃџрЃљ рЃАрЃ«рЃЋрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА рЃЕрЃЋрЃћрЃюрЃА рЃЋрЃћрЃЉ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ. 

рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ html рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃљрЃАрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃљ:

```html copy
<!DOCTYPE html>
<html>
</html>
```

рЃљрЃџрЃЉрЃљрЃЌ рЃўрЃцрЃўрЃЦрЃарЃћрЃЉрЃЌ рЃфрЃљрЃарЃўрЃћрЃџрЃў HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃарЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃљрЃЮ рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃљрЃџрЃўрЃф рЃўрЃЦрЃюрЃћрЃЉрЃўрЃЌ, рЃљрЃарЃљрЃЋрЃўрЃю рЃЦрЃЏрЃюрЃўрЃА HTML рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃА рЃўрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА рЃарЃЮрЃЏ рЃерЃўрЃњрЃюрЃўрЃЌ рЃљрЃарЃљрЃцрЃћрЃарЃў рЃЕрЃљрЃгрЃћрЃарЃЮрЃА. рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃюрЃЋрЃўрЃ«рЃўрЃџрЃЮрЃЌ head рЃЊрЃљ body рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў.

## head рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў
рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў html рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃарЃљрЃАрЃљрЃф рЃЏрЃЮрЃЋрЃљрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃЌ рЃљрЃарЃўрЃА head рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў.

рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў head рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃЌ рЃљрЃАрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃљ:

```html copy lines={3-4}
    <head>
    </head>
</html>
```

head рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃерЃў рЃЉрЃћрЃЋрЃарЃюрЃљрЃўрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃЮрЃЌрЃљрЃЋрЃАрЃЊрЃћрЃА. рЃЌрЃБрЃЏрЃфрЃљ рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃЮ рЃарЃЮрЃЏ рЃЏрЃљрЃАрЃерЃў рЃЌрЃљрЃЋрЃАрЃЊрЃћрЃЉрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў/meta рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃюрЃљрЃЊ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.

### title рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў (head - рЃўрЃА рЃерЃЋрЃўрЃџрЃў)
title рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў - рЃљрЃЊрЃњрЃўрЃџрЃў рЃАрЃљрЃЊрЃљрЃф рЃЋрЃгрЃћрЃарЃЌ рЃЕрЃЋрЃћрЃюрЃў рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃА. 
рЃљрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў рЃљрЃа рЃЕрЃљрЃюрЃА рЃЌрЃљрЃЋрЃљрЃЊ рЃЋрЃћрЃЉ рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃА рЃљрЃарЃћрЃерЃў (viewport - рЃерЃў) рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЕрЃљрЃюрЃА рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃўрЃА рЃбрЃљрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ рЃўрЃАрЃћ рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃЏ рЃАрЃБрЃарЃљрЃЌрЃќрЃћрЃљ рЃюрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃў:

### рЃарЃљрЃбрЃЮрЃЏ рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЕрЃћрЃЏрЃА HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃерЃў рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃў?
**рЃЌрЃБ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃерЃў рЃЦрЃљрЃарЃЌрЃБрЃџ рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ, рЃўрЃАрЃћ рЃарЃЮрЃњрЃЮрЃарЃф рЃќрЃћрЃЏрЃЮрЃЌ title рЃбрЃћрЃњрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃАрЃљрЃЌрЃљрЃБрЃарЃў, рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃдрЃЏрЃЮрЃљрЃЕрЃћрЃюрЃЮрЃЌ рЃарЃЮрЃЏ рЃњрЃљрЃБрЃњрЃћрЃЉрЃљрЃарЃў рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃў рЃЊрЃљрЃўрЃ«рЃљрЃбрЃћрЃЉрЃљ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ. рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃЋрЃљрЃАрЃгрЃЮрЃарЃЮрЃЌ?**

рЃљрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃљ head рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃерЃў рЃЎрЃўрЃЊрЃћрЃЋ рЃћрЃарЃЌрЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЕрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮрЃЌ рЃарЃўрЃЌрЃљрЃф рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃА рЃЏрЃўрЃЋрЃљрЃюрЃўрЃерЃюрЃћрЃЉрЃЌ рЃарЃЮрЃЏ рЃўрЃАрЃћрЃЌрЃў рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃЌрЃљ рЃЊрЃўрЃљрЃърЃљрЃќрЃЮрЃюрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃњрЃЋрЃАрЃБрЃарЃА рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЦрЃљрЃарЃЌрЃБрЃџ рЃљрЃАрЃЮрЃћрЃЉрЃАрЃљрЃф рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА - рЃљрЃАрЃћрЃЌрЃўрЃљ рЃљрЃарЃўрЃА UTF-8. 

рЃљрЃЏ рЃАрЃљрЃЦрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА meta рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЊрЃљрЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃўрЃА рЃљрЃАрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃљ:
`<meta charset="UTF-8">`

рЃљрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃЏрЃљ рЃљрЃа рЃЊрЃљрЃњрЃљрЃЉрЃюрЃўрЃЮрЃЌ, рЃљрЃЏ рЃћрЃбрЃљрЃърЃќрЃћ рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃЮрЃЌ head рЃбрЃћрЃњрЃерЃў рЃЊрЃљ рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ рЃЌрЃљрЃЋрЃћрЃЉрЃерЃў рЃњрЃљрЃюрЃЋрЃўрЃ«рЃўрЃџрЃљрЃЋрЃЌ рЃЏрЃљрЃА рЃБрЃцрЃарЃЮ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ.

рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ рЃЕрЃЋрЃћрЃюрЃў HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃЊрЃў рЃљрЃАрЃћрЃЌрЃў рЃњрЃљрЃЏрЃЮрЃЋрЃљ:
