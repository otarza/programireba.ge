# HTML рЃбрЃћрЃњрЃћрЃЉрЃў (tags)
**рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ HTML рЃбрЃћрЃњрЃћрЃЉрЃўрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃњрЃљрЃю.**

## HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ
рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃА рЃљрЃЦрЃЋрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃарЃЮрЃЏрЃџрЃўрЃЌрЃљрЃф рЃљрЃЏ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃў рЃерЃўрЃюрЃљрЃљрЃарЃАрЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ рЃљрЃю рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃў. 

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃљрЃю Word - рЃўрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃерЃў рЃњрЃЋрЃ«рЃЋрЃЊрЃћрЃЉрЃљ рЃАрЃљрЃЌрЃљрЃБрЃарЃўрЃА, рЃЦрЃЋрЃћрЃАрЃљрЃЌрЃљрЃБрЃарЃўрЃА, рЃљрЃЉрЃќрЃљрЃфрЃўрЃА, рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃўрЃА рЃЊрЃљ рЃљ.рЃе. рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў. рЃљрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ. 

рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃўрЃЌ, HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃАрЃљрЃф рЃљрЃЦрЃЋрЃЌ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃЊрЃљ рЃЋрЃћрЃЉ рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃўрЃА рЃЎрЃћрЃЌрЃћрЃЉрЃљ рЃАрЃўрЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃћрЃерЃў рЃљрЃЏ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ рЃљрЃњрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃљ.

рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃњрЃљрЃю. рЃљрЃЦрЃљрЃф рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃљрЃ«рЃўрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў - рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ h1 рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃљрЃЌрЃљрЃБрЃарЃўрЃАрЃЌрЃЋрЃўрЃА, p рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃљрЃЉрЃќрЃљрЃфрЃўрЃАрЃЌрЃЋрЃўрЃА, a рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЉрЃЏрЃБрЃџрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ, img рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЋрЃћрЃЉ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃЕрЃљрЃАрЃљрЃАрЃЏрЃћрЃџрЃљрЃЊ, video рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЋрЃўрЃЊрЃћрЃЮрЃА рЃЕрЃљрЃАрЃљрЃАрЃЏрЃћрЃџрЃљрЃЊ рЃЊрЃљ рЃљ.рЃе. рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃ«рЃџрЃљ рЃЏрЃЌрЃџрЃљрЃЊ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃљрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃа рЃўрЃюрЃћрЃарЃЋрЃўрЃБрЃџрЃЮ - рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃўрЃАрЃгрЃљрЃЋрЃџрЃў рЃюрЃљрЃЉрЃўрЃ»-рЃюрЃљрЃЉрЃўрЃ».

рЃФрЃљрЃџрЃўрЃљрЃю рЃЉрЃћрЃЋрЃарЃюрЃљрЃўрЃарЃў HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃарЃЮрЃЏрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃф рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃерЃћрЃюрЃў HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА(рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃўрЃА) рЃљрЃАрЃљрЃгрЃДрЃЮрЃЉрЃљрЃЊ рЃЊрЃљ рЃЏрЃљрЃЌрЃў рЃАрЃарЃБрЃџрЃў рЃАрЃўрЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃюрЃљрЃ«рЃЮрЃЌ [MDN - рЃўрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃерЃў](https://developer.mozilla.org/en-US/docs/Web/HTML/Element).

рЃЌрЃБрЃЏрЃфрЃљ рЃљрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃќрЃћрЃърЃўрЃарЃљрЃЊ рЃЊрЃљрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃћрЃЉрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃљрЃа рЃљрЃарЃўрЃА. рЃЕрЃЋрЃћрЃю рЃњрЃљрЃЋрЃћрЃфрЃюрЃЮрЃЉрЃўрЃЌ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа рЃЕрЃљрЃўрЃ«рЃћрЃЊрЃљрЃЋ рЃ«рЃЮрЃџрЃЏрЃћ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃерЃў рЃЊрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЏрЃгрЃБрЃЌрЃљрЃА рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃўрЃЏрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљрЃА рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљрЃА рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃљрЃЋ рЃ«рЃЮрЃџрЃЏрЃћ. рЃЊрЃарЃЮрЃЌрЃљ рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў рЃќрЃЮрЃњрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃгрЃћрЃАрЃў рЃќрЃћрЃърЃўрЃарЃљрЃЊрЃљрЃф рЃЊрЃљрЃњрЃљрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃЊрЃћрЃЉрЃљ рЃарЃљрЃЊрЃњрЃљрЃю рЃ«рЃерЃўрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉ рЃ«рЃЮрЃџрЃЏрЃћ.

## рЃарЃљ рЃљрЃарЃўрЃА рЃбрЃћрЃњрЃў?
HTML рЃбрЃћрЃњрЃў рЃљрЃарЃўрЃА рЃўрЃА рЃарЃўрЃАрЃњрЃљрЃюрЃљрЃф рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃЌрЃБ рЃЏрЃўрЃюрЃЊрЃљ рЃарЃЮрЃЏ рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљ рЃАрЃљрЃЌрЃљрЃБрЃарЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮ `<h1>` рЃбрЃћрЃњрЃў. 

рЃљрЃАрЃћрЃЌрЃўрЃљ рЃбрЃћрЃњрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:
