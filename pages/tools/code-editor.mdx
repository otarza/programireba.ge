# კოდის ედიტორი

პროგრამირების სამყაროში კოდის ედიტორები ის პროგრამებია რომლის საშუალებითაც კოდს ვწერთ. ნებისმიერი პროგრამისტი დროის უდიდეს ნაწილს ატარებს კოდის ედიტორში. ეს არის მთავარი სამუშაო იარაღი. 

წლების განმავლობაში უამრავი უამრავი პოპულარული ედიტორი შეიქმნა და განვითარდა. თუმცა დღესდღეობით ყველაზე მეტად გამოყენებადი VS Code - ია რადგან ის მუშაობს ნებისმიერ პროგრამირების ენასთან და მის ირგვლივ იქმნება ძალიან ბევრი იარაღი რომელიც პროგრამისტის ცხოვრებას და საქმიანობას უფრო ეფექტურს ხდის.
